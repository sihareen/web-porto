Experience Project

PUMMA U-TEWS
Project: PUMMA U-TEWS

Perangkat ini memantau tinggi muka air laut secara real-time di lokasi rawan tsunami. Sistem ini terintegrasi dalam jaringan peringatan dini tsunami nasional Indonesia.

Buoy U-TEWS
Project: Buoy U-TEWS

Buoy U-TEWS adalah pelampung laut yang digunakan di perairan dangkal untuk memantau fluktuasi permukaan laut dan cuaca lokal, mendukung sistem peringatan dini tsunami.

Buoy U-TEWS V2
Project: Buoy U-TEWS V2

Versi lanjutan dari Buoy V1 yang dilengkapi algoritma deteksi tsunami DART. Meningkatkan akurasi dalam pemantauan dan pengambilan keputusan mitigasi bencana.

Automatic Weather Station
Project: Automatic Weather Station

Stasiun iklim yang dirancang untuk memantau kondisi lingkungan di kawasan hutan mangrove. Dilengkapi dengan sensor suhu, kelembaban, tekanan udara, dan intensitas cahaya untuk mendukung penelitian dan konservasi.

Dashboard Climate
Project: Dashboard Climate

Website berbasis Laravel yang menyajikan data hasil pengukuran dari stasiun iklim secara real-time melalui grafik interaktif dan tabel. Mendukung pengambilan keputusan berbasis data lingkungan.

Gas Analyzer
Project: Gas Analyzer

Perangkat ini digunakan untuk mengukur konsentrasi karbon dioksida (CO2) serta parameter lingkungan lainnya. Ideal untuk riset perubahan iklim dan strategi mitigasi karbon di wilayah hutan atau pesisir.

Ini Rizkan
Project: Ini Rizkan

Website portofolio ini dibangun untuk menampilkan profil, keterampilan, dan proyek-proyek saya dengan tampilan yang bersih dan responsif, menggunakan HTML dan CSS murni.

Measure App
Project: Measure App

Perangkat IoT ini memungkinkan pengukuran jarak secara real-time menggunakan sensor ultrasonik, dihubungkan ke smartphone melalui Bluetooth atau Wi-Fi. Cocok untuk aplikasi pemantauan otomatis.

Auto Pump
Project: Auto Pump

Rangkaian simulasi otomatisasi pompa menggunakan sensor IR pada aplikasi Proteus. Sistem ini dapat mengaktifkan pompa saat mendeteksi kehadiran objek dalam jarak tertentu.

Rubus IOC
Project: Rubus IOC

Merancang dan menerapkan Rubus IOC, perangkat pemantauan kelembapan tanah menggunakan LoRa untuk transmisi jarak jauh. Visualisasi data real-time dimungkinkan melalui gateway yang terhubung ke internet. Sistem panel surya dan litium terintegrasi untuk operasi 24/7 di area pertanian terpencil tanpa jaringan listrik, mendukung irigasi berbasis data.
